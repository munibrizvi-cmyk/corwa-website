What you upload

PDFs, scans, faxes, images, and Word docs.

What you get back

  • Readable & searchable document set
  • Clickable Report with visit dates, age/gender, reason, symptoms + duration
  • Vitals/BMI, exam findings, diagnostics, plan/orders
  • Per-page labels (results/HPI/exam/vitals/non-pertinent/blank)

Policy recommendation

  • Clear determination: approvable / deniable / missing items
  • Detailed reasons for each recommendation
  • Everything linked back to the original source page
  • Download as PDF or CSV if needed
